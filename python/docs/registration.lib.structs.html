classmethod projection_matrix(source_position: Tensor, central_ray: Tensor | None = None) → Tensor[source]¶

Generate the projection matrix for the given source position

Parameters:

  • source_position – [(3,) tensor] the position of the X-ray source

  • central_ray – [(3,) tensor or None] the vector from the X-ray source to the closest point on the detector array. If none, the detector array is assumed to be the x-y plane.

Returns:

[(4, 4) tensor] The projection matrix P that projects points in homogeneous coordinates away from the given source position onto the plane of the detector array, as given by the central ray.
